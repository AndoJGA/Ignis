# Ignis Engine
A high-performance 2D Game Engine built from scratch using C++, SDL3, and Vulkan.

## 🎯 Project Goal
Ignis is a technical exploration into low-level engine architecture, memory management, and modern graphics APIs. The objective is to build a robust 2D rendering framework suitable for high-performance applications.

## 🛠️ Current Tech Stack
- **Language:** C++20
- **Windowing/Input:** SDL3
- **Graphics API:** Vulkan 1.4 (In Progress)
- **Build System:** CMake
- **IDE:** CLion

## 🚀 Getting Started
1. Clone the repository.
2. Ensure the Vulkan SDK and a C++20 compatible compiler (MinGW/MSVC) are installed.
3. Open the project in CLion and reload CMake.
